Create a Java program to model an academic institution using OOP concepts like inheritance, encapsulation, and polymorphism. Defi ne a class hierarchy as follows: Staff → Professor → Head of Department (HOD).
A. Staff : Store and display staff name, date of joining, and staff ID.
B. Professor: Inherit all Staff details and additionally display subject specialization and salary.
C. HOD: Inherit all Professor details and additionally display department allowance (if applicable).
OR
Write a program of servlet to demonstrate GET and POST Methods with suitable example using simulation.


Q1

class Staff {

    // Encapsulation
    private String name;
    private String dateOfJoining;
    private int staffId;

    // Constructor
    Staff(String name, String dateOfJoining, int staffId) {

        this.name = name;
        this.dateOfJoining = dateOfJoining;
        this.staffId = staffId;
    }

    // Display Method
    void display() {

        System.out.println("Staff Name: " + name);
        System.out.println("Date of Joining: " + dateOfJoining);
        System.out.println("Staff ID: " + staffId);
    }
}

// Professor Class
class Professor extends Staff {

    private String specialization;
    private double salary;

    Professor(String name, String dateOfJoining, int staffId,
              String specialization, double salary) {

        super(name, dateOfJoining, staffId);

        this.specialization = specialization;
        this.salary = salary;
    }

    // Polymorphism (Method Overriding)
    @Override
    void display() {

        super.display();

        System.out.println("Subject Specialization: " + specialization);
        System.out.println("Salary: " + salary);
    }
}

// HOD Class
class HOD extends Professor {

    private double departmentAllowance;

    HOD(String name, String dateOfJoining, int staffId,
        String specialization, double salary,
        double departmentAllowance) {

        super(name, dateOfJoining, staffId,
                specialization, salary);

        this.departmentAllowance = departmentAllowance;
    }

    // Method Overriding
    @Override
    void display() {

        super.display();

        System.out.println("Department Allowance: "
                + departmentAllowance);
    }
}

// Main Class
public class InstitutionManagement {

    public static void main(String[] args) {

        Staff s1 = new Staff(
                "Rahul",
                "10-01-2020",
                101);

        Professor p1 = new Professor(
                "Isha",
                "15-06-2019",
                102,
                "Java",
                50000);

        HOD h1 = new HOD(
                "Priya",
                "20-03-2015",
                103,
                "Database",
                80000,
                10000);

        System.out.println("\n----- Staff Details -----");
        s1.display();

        System.out.println("\n----- Professor Details -----");
        p1.display();

        System.out.println("\n----- HOD Details -----");
        h1.display();
    }
}
